## Troubleshooting

### "Still getting 500 on POST"
**Cause**: Web admin not reloaded  
**Fix**: Refresh http://localhost:5173 in browser (Ctrl+F5 for hard refresh)

### "POST works but challenge doesn't appear in list"
**Cause**: Page not refreshed after create  
**Fix**: The web admin should auto-refresh via React Query. Try clicking "Next" or refreshing page.

### "Mobile can't see challenge"
**Cause**: Challenge dates might be in the future or past  
**Fix**: When creating, use "today + 7 days" for start/end dates

### "Points not awarded"
**Cause**: Only ACCEPT action awards points, not REJECT or SKIP  
**Fix**: Make sure you clicked "Accept" not "Reject" or "Skip"
