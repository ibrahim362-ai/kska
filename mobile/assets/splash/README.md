# Splash Screens

Place splash screen images here. See `/icon/README.md` for requirements.

Recommended:
- `splash.png` (1242x2436, light mode)
- `splash_dark.png` (1242x2436, dark mode)
- `splash_android12.png` (1152x1152, Android 12+ style)

Generate after placing files:
```bash
dart run flutter_native_splash:create
```
